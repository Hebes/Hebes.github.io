# 启动调试

PC, MAC创建调试工程
    Project Settings设置
    修改 C++ Compiler Configuration为Debug。
    Building Settings中选中 "Create VisualStudio Solution"。
    Build完成后，即产生一个可调试的工程。想了解更多，可参考Unity官方文档
