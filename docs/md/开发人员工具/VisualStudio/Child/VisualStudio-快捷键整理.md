## 替换

![最终效果如下](\../Image/VisualStudio-快捷键整理/1.gif)

因此使用如下组合，就是得到我们想要的结果

```javascript {.line-numbers}
(///\s<.*\n*)(\s*)(///)(\s*)(\w*)(\n*)(\s*///.*)
```

|代码段|正则分组|分组序号|
|----|----|----|----|
|/// \<summary>|(///\s<.*\n*)|$1|
|空格|(\s*)|$2|
|///|(///)|$3|
|空格|(\s*)|$4|
|条形码|(\w*)|$5|
|换行|(\n*)|$6|
|/// \</summary>|(\s*///.*)|$7|

```javascript {.line-numbers}
$1$2$3$4$5$6$7 [Description("$5")]
```
